- Credit Risk Analytics
- Fraud Detection
- AML Monitoring
- Executive Decision Intelligence
- Forecasting & Predictive Analytics
- Explainable AI
- Real-Time Risk Monitoring
- Autonomous Action Engines
- LLM-Powered Banking Copilots

The platform combines:
- Streamlit frontend
- FastAPI backend
- Machine Learning pipelines
- SQLite audit database
- Gemini-powered AI assistants
- Real-time operational intelligence

# ⚡ Installation

## Clone Repository

```bash
git clone https://github.com/AbdoulieJBah/FinGuard-AI-Banking-Risk-Financial-Intelligence-Dashboard.git
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Run Streamlit App

```bash
python -m streamlit run app.py
```

## Run FastAPI Backend

```bash
python -m uvicorn backend_api:app --reload
```

---

# 🔑 Gemini API Setup

Create Streamlit secrets:

```toml
GEMINI_API_KEY = "your_api_key_here"
```

Without an API key, the platform automatically falls back to the built-in executive AI engine.
